# AI-Interview-Agent

## Description

Ai-interview-Agent is an AI-powered autonomous hiring agent that evaluates candidates by comparing resumes with job descriptions and conducting adaptive technical interviews.
It automates the screening and interview process using intelligent question generation and answer evaluation.

---

## Features

*  Resume vs Job Description matching
*  Candidate eligibility screening
*  Adaptive question generation (Basic → Moderate → Hard)
*  AI-based answer evaluation
*  Score calculation for each response
*  Average score generation
*  Final feedback based on overall performance

---

## How It Works

1. User provides Job Description (JD) and Resume
2. System checks if the candidate is eligible
3. If eligible:

   * Generates skill-based questions
   * Asks questions in increasing difficulty (Basic → Moderate → Hard)
4. Evaluates each answer
5. Assigns scores to responses
6. Calculates average score
7. Generates final feedback
